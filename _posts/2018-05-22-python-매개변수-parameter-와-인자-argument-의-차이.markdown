>함수 내부에서 함수에게 전달되어 온 변수는 매개변수라 부르며, 함수를 호출할 때 전달하는 변수는 인자로 부른다.

#### 함수 정의때는 매개변수
def func(매개변수1, 매개변수2):


#### 함수 호출시에는 인자
func(인자1, 인자2)
